## 9. Guia de Prompting Avanzado

### Principios Fundamentales

#### 1. Se Especifico y Claro
```
MAL:  "Hazme un reporte"
BIEN: "Crea un reporte de gastos del Q4 2024 con:
       - Resumen ejecutivo (3 lineas)
       - Tabla de gastos por categoria
       - Comparacion vs Q3 2024
       - 3 recomendaciones de ahorro"
```

#### 2. Proporciona Contexto
```
"Actua como un analista financiero senior en una empresa de pagos fintech.
Tu audiencia es el CFO que necesita datos concisos para presentar al board."
```

#### 3. Define el Formato de Salida
```
"Responde en formato:
1. Titulo
2. Bullet points (maximo 5)
3. Tabla comparativa
4. Conclusion de 2 lineas"
```

#### 4. Usa Ejemplos (Few-shot)
```
"Genera emails de seguimiento como estos ejemplos:
Ejemplo 1: [email de ejemplo]
Ejemplo 2: [otro email]
Ahora genera uno para: [tu caso]"
```

### Templates por Caso de Uso

#### Analisis de Datos
```
Analiza el archivo adjunto y proporciona:
1. Resumen de hallazgos principales (top 3)
2. Tendencias identificadas
3. Anomalias o datos atipicos
4. Recomendaciones accionables
5. Visualizacion sugerida para presentar estos datos

Contexto: [describe tu situacion]
Audiencia: [quien vera este analisis]
```

#### Redaccion de Documentos
```
Escribe un [tipo de documento] sobre [tema] considerando:
- Proposito: [objetivo del documento]
- Audiencia: [quien lo leera]
- Tono: [formal/informal/tecnico]
- Extension: [numero de palabras/paginas]
- Incluir: [secciones requeridas]
- Evitar: [que no incluir]
```

#### Resumenes Ejecutivos
```
Resume el siguiente contenido para [audiencia ejecutiva]:
- Maximo 1 pagina
- Empezar con conclusion principal
- 3-5 puntos clave en bullets
- Incluir: impacto en [metricas relevantes]
- Terminar con: proximos pasos recomendados

[Contenido a resumir]
```

#### Automatizaciones
```
Disena una automatizacion para:
Trigger: [que inicia el proceso]
Pasos: [que debe pasar]
Resultado: [que debe producirse]
Herramientas disponibles: [Make/Zapier/etc]
Integraciones necesarias: [apps a conectar]
```

### Tecnicas Avanzadas

#### Chain of Thought
```
"Piensa paso a paso antes de responder:
1. Primero, analiza el problema
2. Luego, considera las opciones
3. Finalmente, recomienda la mejor solucion
Muestra tu razonamiento."
```

#### Role Prompting
```
"Eres un experto en [campo] con 15 anos de experiencia en [industria].
Tu especialidad es [area especifica].
Responde como lo haria este experto."
```

#### Iteracion
```
"Despues de tu respuesta inicial, revisala y mejorala considerando:
- Es suficientemente claro?
- Faltan detalles importantes?
- La estructura es optima?
Proporciona la version mejorada."
```
